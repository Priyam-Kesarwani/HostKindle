# HostKindle

HostKindle is a full-stack web application that provides a platform for house hosting services, allowing users to sign up either as a **Guest** (to book homes) or as a **Host** (to list properties). The app is built using the **MVC architecture**, promoting separation of concerns and scalable design.

## Features

- Signup and login for two user types: **Guest** and **Host**
- Hosts can list properties with detailed descriptions and images
- Guests can search and book listed accommodations
- Booking management and request system
- Tailwind CSS for responsive and modern UI
- Clean MVC folder structure for scalable code

## Technologies Used

- **Backend**: Node.js, Express.js
- **Frontend**: EJS templates, Tailwind CSS
- **Database**: MongoDB
- **Authentication**: Express-session / Passport (if used)

## Installation

 1. Clone the repository:
    ```bash
   git clone https://github.com/Priyam-Kesarwani/HostKindle.git

 2. Navigate to the project directory:
   ```bash
   cd HostKindle
   ```
 3. Install dependencies:
  ```bash
  npm install
  ```

 4. Start the development server:
  ```bash
  npm start
  ```
