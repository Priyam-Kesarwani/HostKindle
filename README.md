# 🏠 HostKindle

**HostKindle** is a full-stack 🧑‍💻 web application that enables users to either **host** their property 🏡 or **book** a stay as a guest 🎒. Built on the **MVC architecture** for clean and scalable design, it delivers a seamless experience for both parties.

---

## ✨ Features

- 👥 Signup/Login as **Guest** or **Host**
- 📝 Hosts can create property listings with images
- 🔍 Guests can search and book available listings
- 📅 Booking management system for both roles
- 💻 Responsive UI with **Tailwind CSS**

---

## 🛠️ Tech Stack

- ⚙️ **Backend**: Node.js, Express.js
- 🖼️ **Frontend**: EJS templates, Tailwind CSS
- 💾 **Database**: MongoDB
- 🔐 **Authentication**: Express-session 

---

## 📁 Project Structure (MVC)

HostKindle/
├── controllers/ # 💼 Business logic  

├── models/ # 🧠 Data models (User, Listing, Booking)  

├── routes/ # 🌐 API and page routes  

├── views/ # 🖥️ EJS templates  

├── public/ # 🖼️ CSS, images, and JS files  

├── utils/ # 🧰 Helper utilities  

├── app.js # 🚀 Entry point  

├── package.json # 📦 Dependencies  

└── tailwind.config.js # 🎨 Tailwind   


## 👤User Roles

- Guest 🧳 – Browse listings, book stays, view booking history
- Host 🏘️ – List properties, manage guest requests


## Installation

 1. Clone the repository:
    ```bash
   git clone https://github.com/Priyam-Kesarwani/HostKindle.git
    ```

 2. Navigate to the project directory:
   ```bash
   cd HostKindle
   ```
 3. Install dependencies:
  ```bash
  npm install
  ```

 4. Start the development server:
  ```bash
  npm start
  ```
